# Dr.Nut — Responsive Grid Layout Website
Welcome to the Dr.Nut responsive website project!

## 💡 Purpose
This project was created to demonstrate:
- Custom 12-column CSS Grid system  
- Mobile-first responsive design    
- Clean file organization (grid.css + main.css)  
- Multi-page structure (Home + Contact)  

## Installation 💻
No installation required, simply download the zip file and extract.

## Usage
Open index.html in your choice browser.
